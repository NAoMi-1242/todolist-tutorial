
## React Hooksについて

Reactにある便利な機能。使用する際はファイル冒頭に以下の記述が必要。
```tsx
'use client';

　　～～～
```

主に使うものを紹介する。

* **useState**

    ```tsx
    import { useState } from 'react';

    const [ 状態変数, セッター関数 ] = useState< 型 >( 初期値 );
    ```
    変数を管理するHooksであり、セッター関数を使用して状態を設定すると、関連する要素を再レンダリングしてくれる。
    [例はこちら](https://chatgpt.com/share/676d8b9b-1ca4-800a-9a26-6e357b660a7d)

    > ※初期値で型が明らかな場合は、< 型 > はいらない。
    * 初期値が明らか
    ```tsx
    const [count, setCount] = useState(0);

    const [names, setNames] = useState(["牛乳じゃんけん全勝の田中", "写真に写ってるのに合成された佐藤"]);
    ```
    * 初期値が明らかでない
    ```tsx
    const [age, setAge] = useState< number | undefined >();

    const [names, setNames] = useState< string[] >([]);
    ```
    <br>
    セッター関数は現在の値をアロー関数の引数で取得できる。

    ```tsx
    const [names, setNames] = useState< string[] >([]);

    setNames( ["牛乳じゃんけん全勝の田中", "写真に写ってるのに合成された佐藤"] );

    setNames( (currentNames) => [ ...currentNames, "実は入学できていなかった鈴木" ] );
    ```

    <br>

* **useEffect**

    ```tsx
    import { useEffect } from 'react';

    useEffect(() => {

        // 副作用の処理

        return () => {
            // クリーンアップ処理（必要に応じて）
        };
    }, [ 依存配列 ]);
    ```
    初回レンダリング時のみ実行される。[ 依存配列 ] に設定された値が更新されるとサイド実行される。[例はこちら](https://chatgpt.com/share/676d9460-5a54-800a-a61a-991d6a1a8b84)

<br>

* その他

    [こちら](https://qiita.com/seira/items/f063e262b1d57d7e78b4)を参考にしたり調べたりしてね




# React Hooks 一覧

以下は、React標準のHooks、Next.js専用のHooks、および主要なライブラリやユースケースごとのHooksを網羅したリストです。

---

## **1. React 標準 Hooks**
React に標準で含まれる Hooks です。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useState`                 | ローカルな状態を管理します。                                             |
| `useEffect`                | 副作用（データ取得、DOM操作など）を処理します。                         |
| `useContext`               | コンポーネントツリー全体で共有されるコンテキストの値を取得します。        |
| `useReducer`               | 複雑な状態管理やロジックのための代替手段を提供します。                  |
| `useCallback`              | 関数のメモ化を行い、無駄な再レンダリングを防ぎます。                    |
| `useMemo`                  | 計算結果をメモ化してパフォーマンスを向上させます。                       |
| `useRef`                   | DOM要素やミューテーションを参照します。                                 |
| `useLayoutEffect`          | DOMの更新後、描画前に実行される副作用を処理します。                     |
| `useImperativeHandle`      | `ref` をカスタマイズして子コンポーネントのAPIを制御します。              |
| `useDebugValue`            | カスタムフックのデバッグ情報を表示します。                              |
| `useDeferredValue`         | 値の更新を遅延させ、スムーズなUI更新を実現します。                       |
| `useTransition`            | トランジションの状態を管理し、非同期処理中のUIを調整します。             |
| `useId`                    | SSRとクライアント間で一貫性のある一意のIDを生成します。                  |
| `useSyncExternalStore`     | 外部ストアと状態を同期します（ライブラリ作者向け）。                     |
| `useInsertionEffect`       | CSSやDOMノードの挿入前に副作用を処理します（特殊用途）。                 |
| `use`                      | 非同期データの直接取得と待機を行います（React 18+、Suspense向け）。      |

---

## **2. Next.js 専用 Hooks**
Next.js で提供される特有の Hooks です。

| **フック名**                   | **説明**                                                                 |
|--------------------------------|-------------------------------------------------------------------------|
| `useRouter`                   | クライアントサイドでルーティングを管理します。                           |
| `useSearchParams`             | URL のクエリパラメータを取得します。                                     |
| `usePathname`                 | 現在のパス名を取得します。                                               |
| `useSelectedLayoutSegments`   | 現在のレイアウトセグメントを取得します。                                 |
| `useActionState`              | サーバーアクションに関連する状態を管理します。                           |

---

## **3. React Query（TanStack Query） Hooks**
データフェッチングやキャッシュ管理を行うためのライブラリです。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useQuery`                 | データを取得し、キャッシュを管理します。                                 |
| `useMutation`              | サーバーへのデータ送信や更新を行います。                                 |
| `useQueryClient`           | クエリクライアントのインスタンスを取得します。                           |
| `useInfiniteQuery`         | 無限スクロールのためのデータ取得を行います。                             |

---

## **4. React Hook Form Hooks**
フォーム管理とバリデーションを容易にするライブラリです。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useForm`                  | フォームの管理を行います。                                              |
| `useFieldArray`            | フィールドの配列を操作します。                                          |
| `useWatch`                 | フォームのフィールドの値を監視します。                                   |
| `useFormContext`           | フォームのコンテキストを取得します（ネストしたフォーム用）。              |

---

## **5. Recoil Hooks**
グローバル状態管理ライブラリの Recoil で使用される Hooks です。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useRecoilState`           | Recoil 状態を読み取り、更新します。                                      |
| `useRecoilValue`           | Recoil 状態の値を読み取ります（更新不可）。                              |
| `useSetRecoilState`        | Recoil 状態の更新関数を取得します。                                      |
| `useResetRecoilState`      | Recoil 状態をリセットします。                                            |
| `useRecoilCallback`        | Recoil 状態を含む非同期ロジックを実行します。                            |

---

## **6. Jotai Hooks**
Jotai（軽量な状態管理ライブラリ）で使用される Hooks です。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useAtom`                  | Jotai の基本状態を管理します。                                           |
| `useAtomValue`             | Jotai の値を読み取ります（読み取り専用）。                               |
| `useSetAtom`               | Jotai の更新関数を取得します。                                           |

---

## **7. その他の主要 Hooks**
特定のライブラリやアニメーション、ドラッグ＆ドロップで使用される Hooks。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useSpring`（react-spring） | アニメーションのための状態を管理します。                                |
| `useDrag`（react-dnd）      | ドラッグ&ドロップ操作を実現します。                                     |
| `useDrop`（react-dnd）      | ドロップ可能なエリアを定義します。                                       |

---

## **8. カスタムフックの補助 Hooks**
ライブラリ開発者や高度なユースケースで役立つ標準フック。

| **フック名**                | **説明**                                                                 |
|-----------------------------|-------------------------------------------------------------------------|
| `useImperativeHandle`      | 子コンポーネントの `ref` をカスタマイズします。                         |
| `useSyncExternalStore`     | 外部ストアと状態を同期します（高度な用途）。                             |
| `useInsertionEffect`       | CSS挿入前に副作用を処理します（特定ケース用）。                         |
